Code

To use popover.js, you'll need to

  • Import the popover.js library and it's dependancies. 
    <script src="https://unpkg.com/@popperjs/core@2"></script>
<script src="https://news-teller.github.io/media-laboratory-web/popperjs.js"></script>
  • Use the following attributes to define a trigger element using the data-card-id attribute to set guideline card content. 
    <span data-toggle="popover" data-card-id="2">hover to toggle popover</span>
  • You can also customize the design of the popovers with the following classes: .popover, .popover-header, .popover-body.
    See Media Laboratory website for more information.

Bootstrap version

popover.bootsrap.js is a lighter version designed to work with Bootstrap (v4 or v5).

  • Include both Bootstrap's CSS and JS bundle. For more information visit the official documentation.
  • Import the popover.bootsrap.js library. 
    <script src="https://news-teller.github.io/media-laboratory-web/popover.bootsrap.js"></script>
  • Just add the data-card-id attribute to triggers elements, such as: 
    <button type="button" class="btn btn-lg btn-danger" data-bs-toggle="popover"
  title="default title" data-bs-content="default content" data-card-id="15">
    Click to toggle popover
</button>
  • Bootstrap title and data-content (or data-bs-content in version 5.x) will be populated with Media Laboratory guideline cards data.
